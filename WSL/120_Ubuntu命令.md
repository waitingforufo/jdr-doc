# 更改用户密码
### 命令: passwd
root和非root都可以使用. 
passwd命令通过修改 **/etc/shadow** 文件来工作,该文件包括所有账户信息,包括加密密码.

语法:
```bash
sudo passwd [username]
```

当以超级用户或root权限("sudo")执行时,此命令提示输入指定用户的新密码.
没有强制参数,但有一些可选参数可用于修改 passwd 命令的工作方式.

例如: 
 - -d  删除用户的密码
 - "-S": displays information about a user's password status.  
           参数提供用户密码状态的概述。它显示有关密码复杂性、老化、过期日期以及为特定用户配置的其他选项的几行。
 - "-l": locks or disables an account.  
           参数用于锁定或禁用帐户。当此参数与用户名一起用作参数时，该用户将无法再登录，直到他们被解锁。

# 如何重置忘记的密码？
忘记密码可能会让人感到压力，但 Ubuntu 为用户提供了一种重置密码并重新访问其帐户的方法。方法如下：
```bash
    1. 首先在登录屏幕上错误输入密码三次。
    2. 然后，Ubuntu 将为您提供重置密码的选项。单击“忘记密码？”按钮。
    3. 系统将要求您输入用户名，然后要求您两次输入新密码。
    4. 重新启动计算机并使用新密码登录。
```
