@SpringBootConfiguration：Spring Boot設定クラス。(SpringBootのアノテーション)

　　クラスがSpringBootの設定クラスであることを示す。

　　@Configuration：設定クラスであることを示す。(Springのアノテーション)

　　　　設定クラス ---- 設定ファイル。設定クラスもコンテナー内のコンポーネントである：@Component

　　@EnableAutoConfiguration：自動的に設定する機能を有効に。

　　　　spring時に手動で定義する必要のあった設定を、SpringBootが自動的に定義してくれる。@EnableAutoConfigurationがSpringBootに自動設定を有効に設定させる。

　　@AutoConfigurationPackage：パッケージを自動的に設定するアノテーション。

　　　　@Import(AutoConfigurationPackages.Registrar.class)：

　　　　　　Springの下位アノテーション@Import：コンテナーにコンポーネントを追加するもの。

　　　　メインクラス(@SpringBootApplicationのクラス)所属のパッケージとその配下のパッケージから（再帰的）全てのコンポーネントをスキャンし、コンテナーに追加する。

　　@Import(AutoConfigurationImportSelector.class)：自動設定のセレクターをコンテナーに追加

　　EnableAutoConfigurationImportSelector：コンテナーに追加する設定クラスのselector

　　追加する設定クラスをフルークラス名の形で返され、これらの資材はコンテナーに追加される。

　　自動設定クラス(xxxxAutoConfiguration)はたくさんある：

 

AutoConfigurationImportSelector#getAutoConfigurationEntry()

※これらの自動設定クラスが存在するからこそ、手動で設定ファイルを書く面倒が省くことができる。

※web関連設定ファイル

 

SpringFactoriesLoader.loadFactoryNames(getSpringFactoriesLoaderFactoryClass(), getBeanClassLoader());

2.2 値の書き方

2.2.1 リテラル：普通の値（数字、文字列、ブーリアン型）：

　k: v 値はそのまま書く

　　　　””：ダブルコーテーション：エスケープ処理無し。　例： "zhangsan \n lisi"　＝＞ zhangsan改行lisi

　　　　''：シングル・コーテーション：エスケープ処理あり。　例：'zhangsan \n lisi'　＝＞zhangsan \n lisi
